This archive contains raw data used in the paper "Character Recognition for Greek Squeezes" by Howe et al.

Contents:
PNG Images:  Squeeze images used in the study.
Annotations:  Files containing annotated letter box data (proprietary format)
make_xml.py:  Python script to convert annotations to PageXML format.
train_set.txt:  list of squeezes used for training
validation_set.txt:  list of squeezes used for validation
test_set.txt:  list of squeezes used for testing

If you use this dataset, please include the following citation:
"Character Recognition for Greek Squeezes," Nicholas R. Howe, Feiran Chang, Isabella Falbo, Tajhini Brown, and Aaron Hershkowitz.  Submitted to International Journal Of Document Analysis and Recognition (in review).
